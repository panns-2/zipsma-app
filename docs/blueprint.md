# **App Name**: PannsPay

## Core Features:

- Secure Login: User authentication using a secure Student ID or one-time code for parent login.
- Student Profile View: Display student's full name, class, and a summary of fees in a clear, concise view.
- Fee Summary: Show General Fees (Total Assigned, Amount Paid, Balance) with tap-to-expand breakdown.  Show Feeding Fees (Amount Paid, Amount Deducted, Balance, Arrears).
- Overall Summary: Provide an overall summary of total fees (General + Feeding), total paid, and total outstanding balance.
- Contact Options: In-app call, email and Whatsapp buttons.

## Style Guidelines:

- Primary color: Deep blue (#04396d), as requested, to convey a sense of trust, professionalism, and security, befitting an educational financial app. This echoes the color in the school logo (https://i.imgur.com/XvZ5ly7.png).
- Background color: Very light blue (#EBF4FB).  It should be of similar hue to the primary color, but highly desaturated.
- Accent color: A muted, darker blue-purple (#45519B), offering good contrast, to draw attention to important interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font, for clear readability and a modern yet accessible aesthetic.
- Use silhouette icons for clean, simple navigation. Integrate the school's logo (https://i.imgur.com/XvZ5ly7.png) in the blue circle.
- Use color-coded balances: green for paid, red for owing. This lets users quickly view the payment status.
- Use subtle transitions and animations when navigating between sections to improve user experience.
